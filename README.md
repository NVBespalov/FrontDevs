# Front-Devs SPA
The Front Devs app

## Features
- Catalog
- Product

##System Requirements
Before installing the dependencies, make sure your system has the correct Node and Npm versions, otherwise you will get errors.

- Node 6.x.x
- Npm 3.x.x

## Setup

```
$ npm install
```

## Running

```
$ npm start
$ npm run FakeDb
```

## Build

```
$ npm run build
```
